01. About Me

Hello! My name is May and I enjoy creating things with my code on the internet. My interest in web development and data science started in college when I was taking my first Data Analysis Class, programming in the language R. I found interest in the speed in which I can perform analysis using R and Python together.

Fast forward to today, I started my master's program in Data Science in the University of Colorado Boulder and branch out to new technologies, such as SQL.
